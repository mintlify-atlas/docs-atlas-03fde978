## Version highlights

<AccordionGroup>

  <Accordion title="ROCm 7.2.1 — March 25, 2026">

    - Ubuntu 24.04.4 support added (kernel 6.8 GA, 6.17 HWE).
    - JAX 0.8.2 compatibility enabled.
    - hipBLASLt performance improvements for MXFP8 and MXFP4 GEMMs.
    - HIP runtime stability fixes for stream capture, `hipEventQuery`, AQL packet doorbell, and memory-object reference counting.
    - ROCm Offline Installer Creator discontinued; replaced by the ROCm Runfile Installer.
    - New ROCm glossary added to documentation.
    - Updated components: AMD SMI 26.2.2, HIP 7.2.1, hipBLASLt 1.2.2, rocDecode 1.7.0, rocJPEG 1.4.0, rocSHMEM 3.2.0, RPP 2.2.1.

  </Accordion>

  <Accordion title="ROCm 7.2.0 — January 21, 2026">

    - New Node Power Management (NPM) APIs and CLI in AMD SMI.
    - PPT1 power limit support added to AMD SMI.
    - New HIP library APIs: `hipLibraryEnumerateKernels`, `hipKernelGetLibrary`, `hipKernelGetName`, and others.
    - HIP graph node scaling with optimized doorbell ring mechanism.
    - Composable Kernel 1.2.0: mixed-precision fp8×bf8 GEMM, block scaling, pooling, top-k sigmoid, and WMMA gfx12 support.
    - ThinLTO enabled for ROCm compilers via `-foffload-lto=thin`.
    - HIPCC deprecated; AMD Clang recommended for direct invocation.
    - MIGraphX 2.15.0: MXFP4 support, Flash Decoding, dynamic shape improvements.
    - hipTensor 2.2.0: software-managed plan cache, C API headers, moved to rocm-libraries monorepo.
    - HIPIFY 22.0.0: partial CUDA 13.0.0 support, cuDNN 9.14.0, LLVM 21.1.6.

  </Accordion>

  <Accordion title="ROCm 7.x series">

    ROCm 7.0 introduced the 7.x generation with significant updates to the compiler toolchain, library ecosystem reorganization into the `rocm-libraries` monorepo, and expanded support for AMD Instinct MI350 Series GPUs. The 7.x series continues to receive active development.

  </Accordion>

  <Accordion title="ROCm 6.x series">

    The ROCm 6.x series spanned from December 2023 (6.0.0) through August 2025 (6.4.3). Key milestones include initial AMD Instinct MI300 Series support (6.0), expanded math library coverage, and the introduction of ROCprofiler-SDK as the modern profiling infrastructure.

  </Accordion>

  <Accordion title="ROCm 5.x series">

    The ROCm 5.x series ran from February 2022 through October 2023. This generation established the HIP ecosystem, expanded CUDA compatibility tooling via HIPIFY, and introduced foundational versions of the math libraries (rocBLAS, rocSOLVER, rocSPARSE). All 5.x releases have reached end of life.

  </Accordion>

</AccordionGroup>

## Version lifecycle policy

<Note>
AMD does not publish a formal LTS (Long-Term Support) designation for ROCm releases. Patch releases (e.g., 7.2.1) are issued to address critical bugs and OS compatibility updates within a minor release line.
</Note>

- **Active:** The release receives bug fixes, security updates, and OS compatibility patches.
- **EOL (End of Life):** The release no longer receives updates. Users should upgrade to an active release.

Patch releases (the third version number, e.g., `.1` in `7.2.1`) typically address:
- Critical bug fixes in individual components.
- New OS or kernel version support.
- Compatibility updates for dependent frameworks (e.g., JAX, PyTorch).

Minor releases (e.g., `7.2.0`) introduce new features, new hardware support, and API additions. Major releases (e.g., `7.0.0`) may include breaking changes to APIs, compiler requirements, or packaging structure.
