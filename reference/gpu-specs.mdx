The following tables provide hardware specifications for AMD Instinct GPUs, AMD Radeon PRO and Radeon GPUs, and AMD Ryzen APUs. Specifications include VRAM, compute units, cache hierarchy, register files, and GFXIP versions.

<Note>
For supported ROCm versions per GPU and operating system compatibility, see the [Compatibility matrix](/reference/compatibility-matrix). Not all GPUs listed here are supported in the current ROCm release.
</Note>

For definitions of hardware terms used in these tables, see the [Glossary](/reference/glossary).

## Architecture terminology

| Term | Description |
|------|-------------|
| **CDNA** | Compute DNA — AMD's data center GPU architecture series optimized for HPC and AI (Instinct GPUs) |
| **RDNA** | Radeon DNA — AMD's graphics GPU architecture series (Radeon GPUs) |
| **XCD** | Accelerator Complex Die — chiplet containing compute elements on MI300/MI350 series |
| **GCD** | Graphics Compute Die — die containing compute elements on MI100/MI250 series |
| **CU** | Compute Unit — the fundamental programmable execution engine in AMD GPUs |
| **LDS** | Local Data Share — fast on-chip shared memory per compute unit (equivalent to CUDA shared memory) |
| **VGPR** | Vector General-Purpose Registers — per-lane register storage for vector ALU operations |
| **SGPR** | Scalar General-Purpose Registers — per-wavefront register storage for scalar ALU operations |
| **GFXIP** | Graphics IP version — identifies which instruction set and features a GPU generation supports |
