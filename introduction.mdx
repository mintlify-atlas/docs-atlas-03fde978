AMD ROCm is an open-source software platform optimized to extract HPC and AI workload performance from AMD Instinct and AMD Radeon GPUs while maintaining compatibility with industry software frameworks. ROCm provides the full stack: compilers, runtimes, libraries, debuggers, and profilers for GPU-accelerated computing.

## Get started quickly

<Steps>
  <Step title="Check hardware compatibility">
    Verify your AMD GPU is supported. ROCm supports CDNA4/3/2/1 (Instinct) and RDNA4/3 (Radeon) architectures. See the [compatibility matrix](/reference/compatibility-matrix).
  </Step>
  <Step title="Install ROCm">
    Follow the [quick start guide](/quickstart) to install ROCm on Linux using the AMD package repositories.
  </Step>
  <Step title="Verify the installation">
    Run `rocminfo` and `rocm-smi` to confirm ROCm is installed and your GPU is detected.
  </Step>
  <Step title="Start building">
    Write GPU-accelerated code using [HIP](/concepts/programming-models), or get started with [PyTorch on ROCm](/guides/deep-learning-frameworks).
  </Step>
</Steps>

## Core components

ROCm is composed of modular open-source components that you can use independently or together:

| Category | Components |
|---|---|
| Runtime & Compilers | HIP, ROCR Runtime, ROCm LLVM, HIPCC |
| Math Libraries | rocBLAS, rocFFT, rocRAND, rocSOLVER, rocSPARSE |
| ML & AI | MIOpen, MIGraphX, Composable Kernel |
| Communication | RCCL, rocSHMEM |
| Profiling | ROCprofiler-SDK, ROCm Compute Profiler |
| System Management | AMD SMI, rocminfo |
| Debugging | ROCgdb, ROCdbgapi |

<Note>
ROCm is particularly well-suited for GPU-accelerated HPC, AI/ML, scientific computing, and CAD workloads. It supports programming through HIP (CUDA-compatible), OpenCL, and OpenMP.
</Note>
